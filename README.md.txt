Car Sales Dashboard (Tableau Project)

This project presents an interactive car sales dashboard built using Tableau to analyze sales performance and trends.

Objective

The goal of this project is to track key performance indicators such as total sales, average price, and number of cars sold, and to visualize trends over time.

Tools Used
Tableau Public
Excel
Key Features
Year-to-Date (YTD) total sales and growth
Weekly sales trend analysis
Sales distribution by body style and color
Regional sales comparison
Company-wise sales performance
