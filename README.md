# 🚗 Car Sales Dashboard (Tableau Project)

## 📌 Overview

This project presents an interactive Car Sales Dashboard built using Tableau to analyze and monitor sales performance.
It converts raw sales data into clear visual insights, helping identify trends, patterns, and business opportunities.

---

## ❓ Problem Statement

Businesses often lack a centralized system to track sales performance and identify meaningful trends.
This project addresses that gap by providing a dynamic dashboard to monitor key metrics and support data-driven decision-making.

---

## 🎯 Objectives

* Track key performance indicators such as total sales, average price, and number of cars sold
* Analyze sales trends over time
* Identify top-performing regions and product categories
* Understand customer preferences based on car attributes

---

## 🛠️ Tools & Technologies

* Tableau Public
* Microsoft Excel

---

## 📂 Dataset

The dataset includes car sales data with attributes such as:

* Sales revenue
* Car body type
* Color
* Region
* Company/brand

Data cleaning and preprocessing were performed using Excel before building the dashboard.

---

## 📊 Key Insights

* Year-to-Date (YTD) sales analysis highlights overall growth trends
* Peak sales periods identified through weekly trend analysis
* Certain body styles (e.g., SUVs) contribute higher sales compared to others
* Specific regions consistently generate higher revenue
* Customer preferences vary based on car color and type
